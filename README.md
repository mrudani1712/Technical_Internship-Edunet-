# 🏏 Cricket Runs Prediction Using Machine Learning

## 📌 Project Overview

This project was developed as part of the **AICTE – Edunet Foundation Technical Internship**. It uses **Machine Learning** to predict the expected runs a batsman is likely to score based on match conditions. The project also includes an interactive **Streamlit web application** for real-time predictions.

---

## 🎯 Objectives

- Predict the runs scored by a batsman using Machine Learning.
- Analyze historical cricket match data.
- Develop a user-friendly web application for predictions.
- Demonstrate the application of AI in sports analytics.

## 📊 Dataset

The project uses a cricket dataset containing match-related information such as:

- Match ID
- Batsman
- Batting Team
- Bowling Team
- Venue
- Overs
- Wickets
- Runs

The dataset is cleaned and preprocessed before training the machine learning model.

## ⚙️ Machine Learning Algorithm

**Random Forest Regressor**

### Why Random Forest?

- High prediction accuracy
- Handles non-linear relationships
- Reduces overfitting
- Robust and reliable performance

---

## 💻 Streamlit Application

The web application allows users to enter:

- Match ID
- Batsman
- Batting Team
- Bowling Team
- Venue

After clicking **Predict**, the application displays the predicted runs.

Run the application using:

```bash
streamlit run app.py
```
